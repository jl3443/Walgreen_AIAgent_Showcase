# Walgreens Multi-Agent System UI Showcase

基于 Walgreens 品牌色调的智能体系统聊天界面展示，演示 A2A 和 AP2 协议的多智能体交互流程。

## 设计特点

- **Walgreens 品牌色调**：红色 (#E31837) 和金色 (#D4AF37) 渐变设计
- **现代化 UI**：圆角、阴影、平滑动画效果
- **响应式布局**：适配不同屏幕尺寸

## 快速开始

### 前置要求

- Node.js 18+ 
- npm 或 yarn

### 安装

```bash
npm install
```

### 运行开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:3000` 打开。

## 功能特性

- 交互式聊天界面：实时消息展示和输入
- 多智能体可视化：Inventory Agent、Purchase Agent、Orchestrator、AP2 Gateway
- A2A 协议消息流：展示智能体间的消息传递
- AP2 支付授权可视化：支付流程和授权状态
- 实时消息动画：流畅的消息出现效果
- 快速操作按钮：预设常用命令

## 品牌颜色

- 主红色: #E31837
- 深红色: #B8122F
- 金色: #D4AF37
- 米色: #F5E6D3

## 技术栈

- React 18 - UI 框架
- Vite - 构建工具
- Tailwind CSS - 样式框架
- Lucide React - 图标库
